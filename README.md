# Grade Statistics

## Overview
This project is a Java-based Grade Statistics System that processes student marks across multiple sections. It calculates section-level and overall statistics including minimum, maximum, and average values, while applying input validation to ensure data correctness.

The program demonstrates structured programming using modular methods, loops, and basic data aggregation techniques.

## Features
- Processes multiple sections of students
- Input validation for student count and marks
- Calculates section-level statistics (min, max, average)
- Calculates overall statistics across all sections
- Modular method-based design
- Clean console-based output formatting

## How to Run
1. Ensure you have [Java JDK](https://www.oracle.com/java/technologies/downloads/) installed.
2. Clone this repository:
   `git clone https://github.com/abdelhadibarakat/grade-statistics.git`
3. Navigate to the folder:
   `cd grade-statistics`
4. Compile the program:
   `javac GradeStatistics.java`
5. Run the program:
   `java GradeStatistics`
